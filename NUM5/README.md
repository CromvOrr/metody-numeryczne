# Zadanie NUM5
### autor: Oliwier Stręk

Działanie pliku:

    Plik "num5.cpp" rozwiązuje układ równań zadany w zadaniu numerycznym NUM5 oraz 
    znajduje iteracyjnie wektor x dla N = 124.
    
    Używane są metody iteracyjne Jacobiego i Gaussa-Seidela.
    Do pliku wyprowadzane są dane: numer iteracji; norma z różnicy dokładnego rozwiązania i przybliżenia wybraną metodą iteracyjną.
    
    Dokładne rozwiązanie jest obliczane za pomocą biblioteki Eigen.


Uruchamianie:

    Aby uruchomić program należy wpisać komendę "make" lub "make run" (działanie identyczne).
    
    Aby wyczyścić wszystkie pliki wykonywalne należy wpisać "make clean".
