# Zadanie NUM6
### autor: Oliwier Stręk

Działanie pliku:

    Plik "num6.cpp" implementuje metodę potęgową i znajduje największą co do modułu wartość własną macierzy M oraz odpowiadający jej wektor własny. Zastosowywuje również metodę QR bez przesunięć i znajduje wszystkie wartości własne macierzy M.

    Wyniki powyższych algorytmów sprawdzane są poprzez użycie biblioteki Eigen.


Uruchamianie:

    Aby uruchomić program należy wpisać komendę "make" lub "make run" (działanie identyczne).
    
    Aby wyczyścić wszystkie pliki wykonywalne należy wpisać "make clean".
