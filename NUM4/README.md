# Zadanie NUM4
### autor: Oliwier Stręk

Działanie pliku:

    Plik "num4.cpp" rozwiązuje równanie zadane w zadaniu numerycznym NUM4 oraz znajduje wektor y dla N = 80.
    Powyższy wynik sprawdzany jest przez użycie biblioteki Eigen.

    Następnie prowadzona jest symulacja, która sprawdza jak zmienia się czas działania programu względem zmiennej N.
    Dane wyprowadzane są do pliku "dataa.dat".


Uruchamianie:

    Aby uruchomić program należy wpisać komendę "make" lub "make run" (działanie identyczne).
    
    Aby wyczyścić wszystkie pliki wykonywalne należy wpisać "make clean".
